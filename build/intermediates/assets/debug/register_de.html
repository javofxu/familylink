• Details 8226; Geräte- oder App-Informationen, einige mobile Geräte oder Apps enthalten einzigartige App-Nummern.Zum Beispiel die Konfigurationsinformationen, Geräteversion-Nummer, Gerätekennung, IP-Adresse, etc., die vom mobilen Gerät, Browser oder anderen Programmen bereitgestellt werden, die Sie zum Zugriff auf den Heimschutz-Dienst verwenden.Um bessere Dienste zu bieten und die Nutzererfahrung zu verbessern, können wir Informationen wie Hardware-Modell, Betriebssystem-Versionsnummer, internationalen Mobilfunk-Identitätscode (IMEI) aufzeichnen.
• Informationen zum Standort: Wenn Sie die Position des Gerätes einschalten und die entsprechenden Dienste von "Home Guard" auf der Grundlage des Standorts nutzen, verarbeitet das System automatisch die Standortinformationen der relevanten Geräte über GPS oder WLAN usw., so dass Sie relevante Dienste erhalten können, ohne Ihre eigenen geografischen Koordinaten manuell einzugeben. Es wird empfohlen, dass Sie sich mit dem Dienstleister oder Hersteller Ihres Mobilgeräts in Verbindung setzen).
